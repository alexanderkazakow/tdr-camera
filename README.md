# TDR Камера

> Профессиональное мобильное приложение для Android, предназначенное для съемки **RAW-фотографий с истинным динамическим диапазоном (TDR - True Dynamic Range)**.

TDR Camera разработана для фотографов, которым требуется максимально возможный динамический диапазон от мобильного устройства.

Вместо захвата одного RAW-кадра приложение записывает несколько серий с разной экспозицией, состоящих из настраиваемых серий кадров. Каждая серия внутренне объединяется для уменьшения шума сенсора, после чего все уровни экспозиции объединяются в одно **32-битное изображение DNG**, сохраняя детали как в тенях, так и в светлых участках, намного превосходящие возможности обычной RAW-фотографии.

---

## Почему именно истинный динамический диапазон (TDR)?

Динамический диапазон сенсора камеры определяет, сколько деталей может быть одновременно записано в самых темных тенях и самых светлых участках сцены. В реальных условиях динамический диапазон часто превышает возможности одной RAW-съемки смартфона. В результате выбор экспозиции, сохраняющей светлые участки, обычно приводит к шумным или недоэкспонированным теням, а экспозиция для теней часто приводит к обрезанию светлых участков с необратимой потерей информации.

Современная фотография на смартфонах решает эту проблему с помощью вычислительной фотографии. Вместо того чтобы полагаться на одну экспозицию, захватываются несколько RAW-кадров с разными уровнями экспозиции, которые затем объединяются для восстановления информации, недоступной в отдельном кадре. TDR Camera расширяет эту концепцию, захватывая настраиваемые многокадровые брекеты экспозиции. Кадры с одинаковой экспозицией усредняются для уменьшения случайного шума сенсора, после чего все серии экспозиций объединяются в один 32-битный DNG-файл с плавающей запятой. Этот рабочий процесс сохраняет значительно больше восстанавливаемой информации о тенях и светлых участках, чем обычное однокадровое RAW-изображение, при этом поддерживая значительно более низкий уровень шума.

## ✨ Особенности

### 📷 Профессиональная съемка в формате RAW

- Съемка фотографий непосредственно в формате RAW (DNG)
- Ручное управление камерой
- Настраиваемый брекетинг экспозиции
- Разработано для профессиональных рабочих процессов постобработки

### 🌗 Фотография с истинным динамическим диапазоном (TDR)

Приложение создает фотографию со сверхвысоким динамическим диапазоном, используя многоступенчатый конвейер:

1. Захват нескольких серий экспозиций.
2. Суммирование нескольких RAW-кадров внутри каждого уровня экспозиции.
3. Усреднение кадров для уменьшения шума сенсора.
4. Объединение уровней экспозиции в одно HDR-изображение.
5. Экспорт результата в формате DNG с плавающей запятой 32 бит.

---

## ⚙️ Настройка параметров съемки

Пользователь может настроить:

- Количество серий снимков
- Количество кадров, снятых для каждого снимка
- Шаг экспозиции (EV)
- Параметры съемки вручную

Эта гибкость позволяет фотографам оптимизировать качество изображения для различных условий съемки.

## 🎯 Целевая аудитория

TDR Камера предназначена для фотографов, выполняющих профессиональную постобработку RAW в таких программах, как Adobe Camera Raw, Lightroom или других редакторах с поддержкой HDR.

Комбинируя несколько уровней экспозиции и несколько кадров на экспозицию, приложение значительно увеличивает полезный динамический диапазон и одновременно снижает шум изображения.
